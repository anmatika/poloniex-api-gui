# POLONIEX-API-GUI

## init
```` yarn install ````

```` touch server/apikeys.js ````

add these lines to above server/apikeys.js

````module.exports.poloniex_api_key = '<YOUR API KEY>';````
````    module.exports.poloniex_secret = '<YOUR SECRET>';````

## play
```` yarn start ````

browser starts at http://localhost:3000
